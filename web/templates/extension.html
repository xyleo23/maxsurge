Расширение для 2GIS

Userscript для Tampermonkey / Violentmonkey. Парсит 2GIS у вас в браузере и шлёт лиды прямо в MaxSurge. Обходит блокировку серверного IP — работает с вашего компьютера от вашего имени.

Скачать userscript
  1. 1. Установите Tampermonkey:
    Chrome · Firefox · Edge
  2. 2. Нажмите кнопку ниже — Tampermonkey попросит установить скрипт.
    📥 2GIS 📥 Avito
  3. 3. Откройте 2gis.ru, найдите нужные компании.
  4. 4. Нажмите фиолетовую кнопку ⚡ В MaxSurge внизу справа.
  5. 5. Лиды появятся в разделе Лиды.
REST API (для собственных интеграций)

Используйте Bearer-токен в заголовке Authorization: Bearer <api_key>

GET /api/v1/ingest/ping
curl https://maxsurge.ru/api/v1/ingest/ping \
  -H "Authorization: Bearer {{ user.api_key }}"
POST /api/v1/ingest/leads
curl -X POST https://maxsurge.ru/api/v1/ingest/leads \
  -H "Authorization: Bearer {{ user.api_key }}" \
  -H "Content-Type: application/json" \
  -d '{"leads":[{"name":"ООО Ромашка","phone":"+79991234567","city":"Москва"}]}'
