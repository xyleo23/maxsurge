SMS-авторизация отключена MAX. Используйте QR: отсканируйте код из мобильного приложения MAX, где вы уже залогинены.

Как отсканировать?
  1. Откройте MAX на телефоне где вы залогинены
  2. Настройки → MAX на компьютере / Устройства
  3. Нажмите "Подключить устройство" / "Сканировать QR"
  4. Наведите камеру на QR-код

Для уже подключённых аккаунтов или экспортированных из другого софта. Формат совместим с PyMax / Max Sheiker Manager.

PyMax/Max Sheiker формат: SQLite с таблицей auth (token, device_id).

Как добавить аккаунт

⚠ Важно

MAX отключил авторизацию по SMS. Теперь работают только QR login и импорт готовых токенов.

📱 QR login

Самый простой способ. У вас должен быть МАХ на телефоне. Откройте MAX → Настройки → "Устройства" или "MAX на компьютере" → Сканировать QR.

🔑 Токен

Для аккаунтов экспортированных из другого софта (login_token + device_id).

📂 Session file

SQLite файл session.db от PyMax или Max Sheiker Manager. Содержит token + device_id.

📦 Bulk CSV

Массовый импорт купленных аккаунтов. Столбцы: phone, login_token, device_id, proxy.

⚠ Не превышайте 30-50 сообщений в день на аккаунт. Используйте мобильный прокси (RU/KZ/AM beeline/altel) для обхода блокировок.
